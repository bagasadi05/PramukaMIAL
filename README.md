# Pramuka MIAL

Website statis materi Pramuka MI Al Irsyad Al Islamiyyah Kota Madiun.

## Struktur Folder

- `index.html` - halaman awal/beranda. Browser akan membuka file ini sebagai pintu masuk utama.
- `pages/` - semua halaman materi dan latihan.
- `assets/` - CSS, JavaScript, dan gambar pendukung.
- `docs/` - catatan desain dan file referensi, bukan halaman utama.
- `404.html` - halaman fallback jika alamat tidak ditemukan.

## Materi Utama

- Kelas 1-2: sejarah Pramuka, Dwisatya dan Dwidarma, sandi angka, kompas.
- Kelas 3: sejarah Pramuka, Dwisatya dan Dwidarma, sandi angka, kompas.
- Kelas 4: sejarah Pramuka, kompas, sandi A-Z.
- Kelas 5-6: sejarah Pramuka, kompas, sandi rumput, misi gabungan.

## Cara Upload

1. Buka layanan hosting statis yang dipakai.
2. Pilih deploy manual, drag and drop, atau upload folder.
3. Upload seluruh folder `PramukaMIAL`, bukan hanya isi `pages/`.
4. Pastikan `index.html`, `404.html`, `pages/`, dan `assets/` ikut ter-upload.

Tidak ada proses build dan tidak perlu `npm install`.
